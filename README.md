## ✨ Overview

A modern, responsive, and professional **portfolio website** built to showcase my skills, experience, and projects as a **Junior Full-Stack Web Developer (MERN)**.

This portfolio highlights my technical expertise and featured projects with a clean UI, smooth animations, glassmorphism effects, and an interactive design — all crafted with attention to detail.

> 🔗 **Live:** [nishat-jahan-portfolio.netlify.app](https://nishat-jahan-portfolio.netlify.app/)

---

## 🎯 Sections

| Section | Description |
|---|---|
| 👩‍💻 **Hero** | Introduction with animated particle background, role chip, and social links |
| 🙋 **About** | Background, journey, interests, and what drives me |
| 🛠️ **Skills** | Technologies categorized in a terminal/codebase style UI |
| 🎓 **Education** | Academic background and certifications |
| 💼 **Experience** | Professional work history with timeline layout |
| 🚀 **Projects** | Featured projects with live links, GitHub repos, and detail pages |
| 📞 **Contact** | Email form, social links, and direct messaging options |
